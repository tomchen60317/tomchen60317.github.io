使用 HEXO(一): 建立自己的第一個部落格

Posted on 2016-11-18 Edited on 2020-01-20 In Hexo

前言

這是人生的第一篇技術文章，今天主要是介紹如何使用 Hexo 建立自己的 Blog

Hexo 是什麼?

Hexo 是基於 NodeJS 開發的套件，將 Markdown 轉成 Html

安裝與啟動

Step1 安裝 Git 與 NodeJS:

要安裝 Hexo 需先安裝 Git 與 NodeJS
提供官方下載連結 : Git NodeJS

Step2 安裝 hexo-cli :

下指令安裝 hexo-cli

Step3:

安裝完後，就可以使用 “hexo init 部落格名稱” 來建置 Blog

Step4:

接下來剛剛建立的 blog，下指令 “hexo g”

Step5:

接下來下 “hexo s” ，
此時只要在 http://localhost:4000 就可以看到自己的部落格了!!
